## 📌 Description
A computer vision-based system that detects and classifies coin denominations from images using OpenCV. It identifies circular shapes using image processing and classifies coins based on size.

## 🚀 Features
- Detects multiple coins  
- Hough Circle Transform  
- Size-based classification  
- Real-time visualization  

## 🛠️ Tech Stack
- Python  
- OpenCV  
- NumPy
